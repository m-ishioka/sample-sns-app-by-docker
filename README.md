# サンプルSNSアプリ

## 環境情報

- Ruby 3.1.1
- Rails 7.0.4

- 使用ポート
  - db 5432
  - selenium 7900
  - web 3000

## セットアップ

1. `make setup` を実行

## 実行

1. `make app` を実行

## テスト

1. `make test` を実行

## e2eテスト

1. `make test-e2e` を実行

### 画面を表示させた状態でe2eテストの実行

1. `make test-e2e` を実行
2. [セレニウム（クローム）](http://localhost:7900/)を起動
3. 上画面にて、「接続」ボタンを押下
4. パスワードは`secret`を入力

上記の方法で、画面を起動する前にテストが終了する場合は、クロームサーバーを常時起動させた状態でテストを行う

### 留意事項

1. テストはセットアップ後実行のこと
